# node-danmu

### 项目介绍

基于node.js为前端项目vue-danmu提供服务

### vue前端地址

https://gitee.com/bestLuck/vue-danmu

### 项目所使用的框架及插件:
 express 4.17.1
 
 mongoose: 5.9.7
 
 nodemailer: 6.4.6
 
 winston: 3.2.1
### 项目初始化

```
npm install
```

### 运行

```
node ./app.js
```
### 安装nodemon

```
npm i nodemon -g
```

### 自动重启服务

```
nodemon ./app.js
```
